🚀 A full-stack university resource-sharing platform designed to make studying more **collaborative, organized, and efficient**.

Students can browse, download, bookmark, and rate academic resources like PDFs, PPTs, and useful links in a secure environment.

---

## ✨ Key Features

### 🔐 Authentication & Security
- JWT-based secure authentication
- Role-based access control
- Student-only resource access

### 📚 Resource Management (CRUD)
- ➕ Create: Upload and add study materials  
- 📖 Read: Browse, search, and filter by category, semester, and type  
- ✏️ Update: Edit resource details, ratings, and bookmarks  
- ❌ Delete: Secure deletion with authorization

## 🛠️ Tech Stack

### Frontend
- Next.js  
- React  
- TypeScript  
- Tailwind CSS  

### Backend
- NestJS (Node.js / Express-based framework)  
- RESTful API architecture

- PostgreSQL

- Axios

## ⚙️ Backend Highlights

- Secure password hashing using **bcrypt**
- Strong request validation using **class-validator**
- Consistent error handling using **HTTP Exceptions**
